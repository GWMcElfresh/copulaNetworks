# Pairwise Kendall tau deviation from implied prior correlations

Pairwise Kendall tau deviation from implied prior correlations

## Usage

``` r
pairwise_tau_stat(uniformMatrix, impliedCor)
```

## Arguments

- uniformMatrix:

  Matrix of uniform pseudo-observations (n x d).

- impliedCor:

  Prior implied correlation matrix.

## Value

Mean absolute deviation of sample taus from implied taus.
