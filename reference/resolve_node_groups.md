# Resolve node group labels for plotting

Resolve node group labels for plotting

## Usage

``` r
resolve_node_groups(vars, nodeGroups = NULL)
```

## Arguments

- vars:

  Character vector of variable names.

- nodeGroups:

  Named character vector mapping variable to group, or a function
  accepting variable names and returning group labels. Default: all
  `"Node"`.

## Value

Character vector of group labels.
